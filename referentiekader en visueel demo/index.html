Snel Beginnen

1. Include het CSS Framework

<link rel="stylesheet" href="summa-styles.css">

Voeg dit toe aan de <head> van je HTML bestand.

2. Gebruik de Classes

<button class="btn btn-primary">Klik</button>
<div class="card">...</div>
<div class="grid grid-cols-3">...</div>

Gebruik de voorgedefinieerde classes voor consistente styling.

3. Logo's Toevoegen

<!-- Lichte achtergrond -->
<img src="assets/SUMMA_Logo_RGB_Indigo.png"
     class="summa-logo">

<!-- Donkere achtergrond -->

     class="summa-logo">
